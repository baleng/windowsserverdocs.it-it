# <a name="use-the-my-server-app-to-connect-to-windows-server-essentials"></a>Usare l'App My Server per connettersi a Windows Server Essentials

>Si applica a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

L'App My Server per Windows Server Essentials consentono di connettersi alle risorse e di eseguire semplici attività amministrative nel server Windows Server Essentials dal portatile, PC o dispositivo Surface. In My Server, è possibile gestire utenti, dispositivi e gli avvisi e usare file condivisi nel server. Quando si è offline, è possibile continuare a lavorare con i file aperti di recente in My Server. le modifiche apportate offline vengono sincronizzate automaticamente con il server alla successiva che connessione.  
  
 Se il server è in esecuzione il sistema operativo Windows Server Essentials, usare l'app My Server originale. Se il server è in esecuzione il sistema operativo Windows Server Essentials, è necessario utilizzare l'app My Server 2012 R2 aggiornata. Entrambe le app possono essere scaricate da [App per Windows](https://windows.microsoft.com/windows-8/apps).  
  
> [!TIP]

>  Per accedere alle risorse in Windows Server Essentials dal tuo Windows Phone, usare l'app telefonica My Server (per Windows Server Essentials) o l'app telefonica My Server 2012 R2 (per Windows Server Essentials). Per informazioni sull'app My Server per telefoni, vedere [usare l'app My Server per Windows Phone](Work-Remotely-in-Windows-Server-Essentials.md#BKMK_2). Per informazioni sull'app My Server 2012 R2 per telefoni, vedere il post di blog [App My Server 2012 R2 Windows e Windows Phone](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx).

##  <a name="BKMK_WhatsNew"></a>Novità in My Server 2012 R2  
 Oltre alle funzionalità disponibili in My Server, My Server 2012 R2 offre le nuove funzionalità seguenti per i clienti con Windows Server Essentials:  
  
-   **Accesso alle raccolte e i siti del team di SharePoint Online** -integrazione di utilizzare Microsoft Office 365 con Windows Server Essentials per accedere ai siti del team di SharePoint Online e aprire i documenti nelle raccolte di SharePoint Online in My Server 2012 R2.  
  
-   **Le connessioni Desktop remoto ai computer client e server** -uso **connessione remota** in My Server 2012 R2 per connettersi ai computer client e server Windows Server Essentials tramite Desktop remoto.  
  
##  <a name="BKMK_OS"></a>Requisiti del sistema operativo  
 Per usare My Server o My Server 2012 R2 nel tuo portatile, un PC o un dispositivo Surface, il computer deve disporre di uno dei seguenti sistemi operativi: Windows 8.1, Windows RT 8.1, Windows 8 o Windows RT.  
  
##  <a name="BKMK_Install"></a>Installare My Server  
 È possibile scaricare l'app My Server appropriata dal [App per Windows](https://windows.microsoft.com/windows-8/apps) archiviare. Se il computer è il sistema operativo Windows 8 o Windows RT e si desidera connettersi al server nella intranet con il nome del server, è necessario installare un certificato dal server.  
  
#### <a name="to-install-my-server-or-my-server-2012-r2-on-your-windows-based-laptop-pc-or-surface-device"></a>Per installare My Server o My Server 2012 R2 nel portatile basato su Windows, PC o dispositivo Surface  
  
1.  Installare l'app My Server appropriata da [App per Windows](https://windows.microsoft.com/windows-8/apps) nel portatile basato su Windows, PC o dispositivo Surface.  
  
    |Sistema operativo server|Download|  
    |-----------------------------|--------------|  
    | Windows Server Essentials|[Il Server](https://apps.microsoft.com/webpdp/app/19d94f81-db21-4234-8b49-806694dbbaea)|  
    | Windows Server Essentials|[My Server 2012 R2](https://apps.microsoft.com/webpdp/app/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
  
2.  Se si desidera essere in grado di utilizzare il nome del server per connettersi al server di Windows Server Essentials in una rete intranet e il dispositivo ha il sistema operativo Windows 8 o Windows RT, installare il certificato predefinito CAROOT.cer nel computer eseguendo la procedura seguente. Installazione manuale di un certificato non è necessaria in Windows 8.1 e Windows RT 8.1.  
  
###  <a name="BKMK_InstallCert"></a>Per installare un certificato per My Server nel tuo dispositivo Windows 8, Windows 8.1 o Windows RT  
  
1.  Aprire un web browser nel computer e scaricare il certificato predefinito CAROOT.cer dal server. A tale scopo, digitare il comando seguente, sostituendo il nome del server (ad esempio, **marketing.contoso.com**) per <*nomeserver*>:  
  
     **http://<servername\>/Connect/default.aspx?Get=caroot.cer**  
  
2.  Installare il certificato:  
  
    1.  Nel **Start** pagina, aprire **Esplora File**.  
  
    2.  Assicurarsi che siano visualizzate gli elementi nascosti e le estensioni di file: nel **visualizzazione** nella scheda il **Mostra/Nascondi** gruppo, selezionare il **gli elementi nascosti** e **estensioni di File** caselle di controllo.  
  
    3.  Passare alla cartella che contiene il file CAROOT.cer appena scaricato.  
  
    4.  Pulsante destro del mouse sul file CAROOT.cer e quindi fare clic su **aprire**.  
  
    5.  Nel **certificato** la finestra di dialogo, fare clic su **Installa certificato**.  
  
    6.  Selezionare **macchina locale** come percorso di installazione e quindi fare clic su **Avanti**.  
  
    7.  Nel **archivio certificati** pagina della procedura guidata, seleziona **colloca tutti i certificati nel seguente archivio**e utilizzare **Sfoglia** per scegliere il **autorità di certificazione radice attendibili** archiviare. Fare clic su **fine**.  
  
##  <a name="BKMK_UseServer"></a>Usare My Server  
 Per iniziare a usare l'app My Server o My Server 2012 R2, aprire l'app e guardare la Demo introduttiva delle funzionalità.  
  
#### <a name="to-open-my-server-or-my-server-2012-r2"></a>Per aprire My Server o My Server 2012 R2  
  
1.  Aprire My Server (per Windows Server Essentials) o My Server 2012 R2 (per Windows Server Essentials) dal **Start** pagina del tuo dispositivo Windows.  
  
2.  Accedere al server di Windows Server Essentials con il tuo account nel server. Per identificare il server:  
  
    -   In un ambiente non locale, ad esempio, utilizzare il nome di dominio del server s **marketing.contoso.com**.  
  
    -   Se si sono connessi in locale tramite una rete intranet, usare il nome del server nell'esempio precedente, il nome del server è **marketing**. (Se si utilizza un computer Windows 8 o Windows RT per la connessione locale e si desidera utilizzare questo metodo, è necessario installare un certificato CAROOT.cer dal server. Per ulteriori informazioni, vedere [per installare un certificato per My Server nel tuo dispositivo Windows 8, Windows 8.1 o Windows RT](#BKMK_InstallCert).)

##  <a name="BKMK_ConnectServer"></a>Come connettersi al server dalla rete locale  
 Per usare correttamente l'app My Server in Windows Server Essentials per Windows Phone, Windows 8 e Windows 8.1, è innanzitutto necessario installare il certificato del server nel dispositivo. Questo si connette il dispositivo al server che esegue Windows Server Essentials nella rete locale. Utilizzare le procedure seguenti per connettersi al server nella rete locale e installare il certificato del server nel tuo Windows Phone o in un computer che esegue Windows 8 o Windows 8.1.  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>Per connettersi al server dal tuo Windows Phone  
  
1.  Nel tuo Windows Phone che esegue Windows 8 o Windows 8.1, aprire Internet Explorer.  
  
2.  Nella barra degli indirizzi digitare **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, quindi premere INVIO.  
  
3.  Quando viene chiesto di installare il certificato caroot.cer, fare clic su **installare**.  
  
4.  Attendere completare l'installazione del certificato.  
  
    > [!NOTE]
    >  Una volta completata l'installazione del certificato, è possibile accedere per l'app My Server per Windows Phone con il nome del server e le credenziali di rete.  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>Per connettersi al server nella rete locale da un computer che esegue Windows 8 o Windows 8.1  
  
1.  In un computer che esegue Windows 8 o Windows 8.1, aprire Internet Explorer.  
  
2.  Nella barra degli indirizzi digitare **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, quindi premere INVIO.  
  
3.  Quando viene chiesto di installare il certificato caroot.cer, fare clic su **aprire**.  
  
4.  Nel **certificato** la finestra di dialogo, fare clic su **installazione certificato**.  
  
5.  In Importazione guidata certificati, selezionare **macchina locale** come percorso dell'archivio.  
  
6.  Selezionare **colloca tutti i certificati nel seguente archivio**, quindi individuare il **autorità di certificazione radice attendibili** percorso.  
  
7.  Seguire le istruzioni per completare la procedura guidata.  
  
    > [!NOTE]
    >  Una volta completata l'installazione del certificato, è possibile accedere all'app My Server per Windows 8 o Windows 8.1 utilizzando il nome del server e le credenziali di rete.
