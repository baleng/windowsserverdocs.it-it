# <a name="checklist-setting-up-a-federation-server-proxy"></a>Elenco di controllo: Configurazione di un proxy server federativo

>Si applica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Questo elenco di controllo include le attività di distribuzione per la preparazione di un server che esegue Windows Server® 2012 per il ruolo proxy server federativo in Active Directory Federation Services \(ADFS\).  
  
> [!NOTE]  
> Completare le attività dell'elenco di controllo nell'ordine indicato. Quando un collegamento a un riferimento porta a una procedura, tornare a questo argomento una volta completati i passaggi della procedura, in modo da poter procedere con le operazioni rimanenti nell'elenco di controllo.
