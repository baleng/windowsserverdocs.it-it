# <a name="attributes"></a>attributi



Visualizza, imposta o Cancella gli attributi di un disco o volume.

## <a name="syntax"></a>Sintassi

```
Attributes disk
Attributes volume
```

## <a name="parameters"></a>Parametri

|Parametro|Descrizione|
|---------|-----------|
|[disco di attributi](attributes-disk.md)|Visualizza, imposta o Cancella gli attributi di un disco.|
|[volume di attributi](attributes-volume.md)|Visualizza, imposta o Cancella gli attributi di un volume.|
