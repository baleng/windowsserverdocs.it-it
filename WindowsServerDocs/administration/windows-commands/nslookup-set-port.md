# <a name="nslookup-set-port"></a>nslookup set port



Consente di modificare la porta predefinita del server dei nomi TCP/UDP Domain Name System (DNS) al valore specificato.

## <a name="syntax"></a>Sintassi

```
set port=<Port>
```
