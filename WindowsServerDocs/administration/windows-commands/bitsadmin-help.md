# <a name="bitsadmin-help"></a>bitsadmin help



Visualizza l'utilizzo della riga di comando.

## <a name="syntax"></a>Sintassi

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>Esempi

Nell'esempio seguente recupera la Guida della riga di comando.
```
C:\>bitsadmin /help
```
