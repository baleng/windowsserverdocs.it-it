# <a name="nslookup-set-vc"></a>nslookup set vc



Specifica se utilizzare o non utilizzare un circuito virtuale durante l'invio di richieste al server.

## <a name="syntax"></a>Sintassi

```
set [no]vc
```

|Parametro|Descrizione|
