# <a name="begin-backup"></a>Avviare il backup



Avvia una sessione di backup completo.

## <a name="syntax"></a>Sintassi

```
begin backup
```

## <a name="remarks"></a>Note

-   Questo comando sostituisce l'impostazione di backup di copia predefinito.
