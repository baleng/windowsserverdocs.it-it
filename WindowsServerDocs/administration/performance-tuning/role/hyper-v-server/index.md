# <a name="performance-tuning-hyper-v-servers"></a>Ottimizzazione delle prestazioni di server Hyper-V


Hyper-V è il ruolo del server di virtualizzazione in Windows Server 2016. I server di virtualizzazione possono ospitare più macchine virtuali che sono isolate l'una dall'altra, ma che condividono le risorse hardware sottostanti grazie alla virtualizzazione dei processori, della memoria e dei dispositivi di I/O. Consolidando i server in un'unica macchina, la virtualizzazione può migliorare l'uso delle risorse e l'efficienza energetica, nonché ridurre i costi operativi e di manutenzione dei server. Le macchine virtuali e le API di gestione inoltre offrono maggiore flessibilità per la gestione delle risorse, il bilanciamento del carico e il provisioning dei sistemi.
