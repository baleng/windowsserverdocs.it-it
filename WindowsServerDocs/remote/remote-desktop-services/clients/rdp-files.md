# <a name="supported-remote-desktop-rdp-file-settings"></a>Impostazioni del file RDP desktop remoto è supportato

La tabella seguente include l'elenco di impostazioni di file RDP supportate che è possibile usare con i client Windows e HTML. Una "x" nella colonna piattaforma indica che l'impostazione è supportata. Tuttavia, questo elenco non è un elenco completo di impostazioni supportate per i client Windows e HTML5. Continueremo aggiornare questa tabella per includere più RDP impostazioni supportate per i client Windows e HTML5, nonché di macOS, iOS e i client Android.
